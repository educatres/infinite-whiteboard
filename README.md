# 無邊際白板 Infinite Whiteboard

一個純 HTML、CSS、JavaScript 實作的無邊際白板，可直接部署到 GitHub Pages，也可以直接用瀏覽器開啟 `index.html` 使用。

## 功能

- 自由筆刷、橡皮擦、直線、長方形、圓形、橢圓形
- 筆畫橡皮擦，可一次刪除整個筆畫或圖形
- 顏色選擇、快速色票與筆刷粗細調整
- 筆跡平滑開關
- 返回上一步，支援按鈕與 `Ctrl/⌘+Z`
- 世界座標儲存筆跡，支援無邊際平移與縮放
- 滑鼠滾輪縮放、空白鍵拖曳平移
- 觸控單指繪圖、雙指平移與縮放
- 匯出目前畫面為 PNG
- 匯出 JSON、匯入 JSON 重建白板
- 無後端、無資料庫、無 npm build

## 使用方式

直接開啟 `index.html`，或將整個專案部署到 GitHub Pages。

桌機操作：

- 滑鼠拖曳：使用目前工具繪圖
- 滾輪：以滑鼠位置為中心縮放
- 空白鍵 + 拖曳：平移白板

觸控操作：

- 單指拖曳：使用目前工具繪圖
- 雙指拖曳：平移白板
- 雙指張合：縮放白板

## 檔案

- `index.html`：白板主程式
- `無邊際白板規格書.md`：開發規格書
