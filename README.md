# 無邊際白板 Infinite Whiteboard

![無邊際白板 Infinite Whiteboard UI](UI.png)

Infinite Whiteboard 是一款專為教學情境設計的無邊際虛擬白板，可直接在電腦、平板、手機或教室電子白板上使用，支援自由書寫、繪圖、加入便條貼，適合用於課程講解、概念整理、腦力激盪、小組討論與重點統整。使用者無須註冊帳號，也不必安裝額外軟體；有網路時可直接開啟網站使用，無網路時只需下載 `index.html` 並以瀏覽器開啟即可執行。白板內容不會上傳至伺服器，也不會留存任何使用資料，所有內容僅存在於目前使用的裝置與瀏覽器中；如需保存，可下載為圖片或匯出白板檔案，並於其他裝置重新匯入後繼續編輯。

直接使用：[https://educatres.github.io/infinite-whiteboard/](https://educatres.github.io/infinite-whiteboard/)

## 功能

- 自由筆刷、橡皮擦、直線、長方形、圓形、橢圓形
- 加入 200×200 便條貼，可選淡黃色、淡綠色、粉紅色、粉藍色並拖曳右下角調整大小
- 筆畫橡皮擦，可一次刪除整個筆畫或圖形
- 筆畫選擇工具，可選取並移動單一筆畫、圖形或便條貼
- 便條貼內建刪除、編輯、確認編輯按鈕
- 刪除所選物件，支援按鈕與 `Delete` / `Backspace`
- 顏色選擇、快速色票與筆刷粗細調整
- 筆跡平滑開關
- 返回上一步，支援按鈕與 `Ctrl/⌘+Z`
- 世界座標儲存筆跡，支援無邊際平移與縮放
- 滑鼠滾輪縮放、空白鍵拖曳平移、右鍵拖曳平移、平移工具單指拖曳
- 觸控單指繪圖、iPad/手機雙指平移與縮放
- iPad 書寫時停筆不觸發瀏覽器長按選單
- iPad/手機手寫效能優化，降低筆跡延遲與重繪負擔
- 自動儲存筆觸，重新整理後仍會保留白板內容
- 匯出目前畫面為 PNG
- 匯出 JSON、匯入 JSON 重建白板
- 無後端、無資料庫、無 npm build

## 使用方式

直接開啟 `index.html`，或將整個專案部署到 GitHub Pages。

桌機操作：

- 滑鼠拖曳：使用目前工具繪圖
- 滾輪：以滑鼠位置為中心縮放
- 空白鍵 + 拖曳：平移白板

觸控操作：

- 單指拖曳：使用目前工具繪圖
- 雙指拖曳：平移白板
- 雙指張合：縮放白板
